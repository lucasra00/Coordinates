Welcome to Waypoint Manager. A program developed to make flying a simpler task when it comes to navigating. You enter, save or load different flights, which can consist of different waypoints. Additionally, you can also add target points, airfields or anything else the you find to be useful during your flight, which has to do with coordinates. This program was developed with DCS F-14 Tomcat in mind, as well as the DCS Mirage-2000C, but can basically be used in any plane any in flight sim.
