Welcome to Waypoint Manager. A program develeped to make flying a simpler task when it comes to navigating. You enter, save or load different flights, which can consist of different waypoints. Additionaly, you can also add target points, airfeilds or anything else the you finde to be usefull dureing you flight, which has do with coordinates. This program was develped with DCS F-14 tomcat in mind, as well as the DCS Mirage-2000C, but can basicly be used in any plane any flight sim.
